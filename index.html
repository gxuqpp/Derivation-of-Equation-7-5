式7-5 推导过程 · 副簧切入频率跳变最小

《汽车设计》第6版 §7.3 · 第二种设计方法

方法一：偏频不变 → c_a/c_m=√λ−1 vs 方法二：跳变最小 → c_a/c_m=(2λ−2)/(λ+3)
1
方法二的设计目标与背景
方法一（式7-4）保证了空满载偏频接近，但在副簧切入点 FK 前后，偏频仍会发生突变跳变（刚度从 cm 突变为 cm+ca）。
⚡
方法一存在的问题：
切入点前：n前 = (1/2π)√(cm·g/FK)
切入点后：n后 = (1/2π)√((cm+ca)·g/FK)
跳变比：n后/n前 = √((cm+ca)/cm) = √√λ = λ1/4（方法一中）
方法二的目标：选取切入载荷 FK 和刚度比，使切入点前后偏频跳变量最小， 对经常处于半载运输状态的货车更为适用。
🎯
设计原则（方法二）：
① FK = (F0 + F∞)/2（空满载的算术平均值）
② 使 [F0, FK] 段平均载荷对应的偏频 = [FK, F∞] 段平均载荷对应的偏频
2
确定副簧切入载荷 FK
方法二中，切入载荷取空满载的算术平均值（对比方法一的几何平均值）：
方法二 FK = F0 + F∞2
方法一 FK = √(F0·F∞)　（几何平均）
引入载荷比 λ = F∞/F0，设 F0 = 1（归一化），则 F∞ = λ，代入切入载荷：
归一化 F0=1，F∞=λ，FK = 1+λ2
物理含义：FK 恰好位于空满载的中间位置，意味着副簧在"半载"时开始介入，这正好对应经常在半载状态运营的货车的工况。
3
计算两段的平均载荷
以 FK 为界，将工作范围分为两段，分别计算各段的平均载荷：
段① [F0, FK]　仅主簧工作（刚度 cm）
平均载荷 F̄1 = F0 + FK2 = F0 + F0+F∞22 = 3F0+F∞22 = 3F0+F∞4
段② [FK, F∞]　主副簧共同工作（刚度 cm+ca）
平均载荷 F̄2 = FK + F∞2 = F0+F∞2 + F∞2 = F0+3F∞22 = F0+3F∞4
规律：段①平均载荷 = (3F0+F∞)/4，段②平均载荷 = (F0+3F∞)/4。 两者之和 = F0+F∞，均值 = (F0+F∞)/2 = FK，符合对称性。
4
建立偏频相等方程（核心设计条件）
回顾：等效静挠度 fc = F/c 决定偏频（式7-3）。 偏频相等 ⟺ 等效静挠度相等 ⟺ F/c 相等
📐
设计条件： 段①平均载荷处偏频 = 段②平均载荷处偏频
即：n( F̄1, cm ) = n( F̄2, cm+ca )
利用等效静挠度转化：
段① fc = F̄1cm = (3F0+F∞)/4cm
段② fc = F̄2cm+ca = (F0+3F∞)/4cm+ca
令相等 3F0+F∞4·cm = F0+3F∞4·(cm+ca) 　⋯⋯ 方程★
两边同乘以 4，消去分母中的 4：
化简 cm·(F0+3F∞) = (cm+ca)·(3F0+F∞) 　⋯⋯ 方程★
5
求解刚度比 ca/cm
从方程★出发，逐步求解 ca/cm：
①
展开右边：
cm(F0+3F∞) = cm(3F0+F∞) + ca(3F0+F∞)
②
移项，ca 项留在右边：
cm(F0+3F∞) − cm(3F0+F∞) = ca(3F0+F∞)
③
化简左边括号：
(F0+3F∞) − (3F0+F∞) = −2F0+2F∞ = 2(F∞−F0)
④
代回：
cm·2(F∞−F0) = ca·(3F0+F∞)
⑤
两边除以 cm·(3F0+F∞)：
中间结果 cacm = 2(F∞−F0)3F0+F∞
6
用载荷比 λ 表示（式7-5）
分子分母同除以 F0，引入 λ = F∞/F0：
①
分子：2(F∞−F0) ÷ F0 = 2(λ−1)
②
分母：(3F0+F∞) ÷ F0 = 3+λ
③
代入，分子展开：
式7-5 · 主副簧刚度比（方法二）
ca/cm = (2λ−2) / (λ+3)
其中 λ = F∞/F0（满载/空载载荷比）
推导来自：切入载荷取算术均值 + 两段平均偏频相等条件
7
数值验证（λ=4，解放货车后轴）
以 λ=4（解放货车后轴典型工况）代入验证：
FK = (F0+F∞)/2 = (1+4)/2 = 2.5 F0
式7-5 ca/cm = (2×4−2)/(4+3) = 6/7 ≈ 0.857
段①均载 F̄1 = (3×1+4)/4 = 1.75 F0
段②均载 F̄2 = (1+3×4)/4 = 3.25 F0
验证偏频相等：
段① fc = F̄1/cm = 1.75/1 = 1.75 F0/cm
段② fc = F̄2/(cm+ca) = 3.25/(1+6/7) = 3.25/(13/7) = 3.25×7/13 = 1.75 F0/cm ✓

8
推导总结 · 完整推导链
式7-5 的推导链条简洁清晰：
A
给定：FK = (F0+F∞)/2（算术均值）
B
计算：F̄1 = (3F0+F∞)/4，F̄2 = (F0+3F∞)/4
C
条件：F̄1/cm = F̄2/(cm+ca)（偏频相等⟺等效静挠度相等）
D
求解：ca/cm = 2(F∞−F0)/(3F0+F∞) = (2λ−2)/(λ+3)
两种方法完整对比
方法一：FK=√(F0F∞)，ca/cm=√λ−1
方法二：FK=(F0+F∞)/2，ca/cm=(2λ−2)/(λ+3)
方法二副簧刚度更小，切入前后频率跳变更小；方法一保证极端工况偏频接近
当λ=1时（空满载相同）：方法一 ca/cm=0，方法二 ca/cm=0，两者退化为线性弹簧，符合物理直觉。
当λ→∞时：方法一 ca/cm→∞，方法二 ca/cm→2，方法二增长更慢，结构更合理。
